Rails template
==============


## Requirement
* postgresql

## Usage
```
# need git-managed directory
mkdir app_name
cd app_name
git init
git remote add origin https://github.com/user_name/repo_name.git
cd ../

# rails new
rails new app_name -m https://raw.githubusercontent.com/taigak/rails-template/master/template.rb
```
